// Technical Challenges Resolved

ZFS Mirror Degraded — Live Repair

Diagnosed a degraded ZFS mirror caused by a vdev label mismatch. Performed a live detach and re-attach of the affected disk to rebuild the mirror without data loss or service interruption.

Cloudflare Bypass — FlareSolverr → Byparr Migration

FlareSolverr became non-functional after Cloudflare updated its anti-bot protections. Replaced with Byparr, which restores Prowlarr's ability to query protected indexers while remaining containerized.

Shared Storage Across 7 LXC Containers

Configured a single ZFS dataset (/bulk/media) as a bind-mounted shared volume accessible to all 7 media-handling containers, eliminating duplicate storage and ensuring consistent file paths across the pipeline.

WireGuard VPN Gateway with Kill Switch

Built a dedicated WireGuard VPN container acting as the network gateway for all download traffic. Configured iptables NAT rules and a kill switch on Proxmox's vmbr1 bridge, ensuring all qBittorrent traffic routes exclusively through ProtonVPN.

// PROJECT 03

Network-Wide DNS Filtering with AdGuard Home

● Live

A self-configured network-wide DNS sinkhole and filtering solution deployed in a dedicated LXC container on Proxmox. All DNS traffic on the network is routed through AdGuard Home, with encrypted upstream resolvers (DoH/DoT) replacing plaintext DNS entirely. Blocklists cover ads, trackers, and malicious domains across every device on the network.

// Service Stack

AdGuard Home
DNS over HTTPS (DoH)
DNS over TLS (DoT)
LXC Container
Proxmox VE

// DNS Architecture

Network devices → Router upstream DNS → AdGuard Home LXC
   └── All DNS queries filtered through sinkhole before resolution
AdGuard Home → Encrypted upstream resolvers
   ├── Cloudflare (DoH / DoT)
   └── Quad9 (DoH / DoT) — malware-blocking resolver
No plaintext DNS leaves the network

// Blocklist Stack

AdGuard DNS Filter  —  Primary ad & tracker blocklist
EasyList  —  Extended ad network coverage
Steven Black Hosts  —  Consolidated hosts-based blocklist

// Technical Challenges Resolved

Encrypted Upstream DNS — Eliminating Plaintext Resolution

Configured AdGuard Home to use DoH and DoT resolvers (Cloudflare, Quad9) as upstream providers, ensuring all DNS queries leaving the network are encrypted. Plaintext UDP/53 queries to external resolvers are no longer used.

Blocklist Stack & Network-Wide Integration

Deployed and tuned multiple blocklists (AdGuard DNS Filter, EasyList, Steven Black hosts) to balance coverage and false positives. Pointed the router's upstream DNS to the LXC container IP, applying filtering across all devices on the network without per-device configuration.
